# Microsoft 365 Digital Transformation — ALEXIS INTERNATIONAL

This repo documents a Microsoft 365 “modern workplace” implementation for a fully remote startup (10-person consulting firm) operating across **Nigeria, America, and India**.

## What’s Included
- Entra ID identities, security groups, MFA, SSPR, PIM
- Teams + Exchange Online (shared mailbox)
- SharePoint departmental sites + migration approach
- Power Automate approval workflow (Finance uploads)
- DLP + Conditional Access + Defender phishing simulation
- OneDrive sync/sharing/version restore
- Power BI dashboard using SharePoint-hosted data
- Admin Units for location-based administration

## Repo Structure
- `docs/` — implementation notes aligned to capstone phases
- `architecture/` — diagrams + source
- `screenshots/` — evidence screenshots (organized by service)
- `configs/` — user/group lists + policy summaries + migration plan
- `templates/` — risk register, testing checklist, rollback plan
- `deliverables/` — final Admin Guide PDF
