# Executive Summary

## Client
ALEXIS INTERNATIONAL — fully remote consulting firm (10 users) across Nigeria, America, and India.

## Goal
Modernize operations by moving from on-prem file shares into Microsoft 365 for collaboration, security, automation, and easier administration.

## Solution Overview
- Entra ID: users, groups, MFA, SSPR, PIM
- Teams + Exchange: departmental Teams, shared mailbox for support
- SharePoint: departmental sites, group-based access, migration approach
- Power Automate: Finance upload approval
- Security: DLP, Conditional Access, Defender phishing simulation
- OneDrive: sync, sharing, version restore
- Power BI: dashboard using SharePoint-hosted Excel
- Admin Units: location-based management
